# 🚀 Qwen 集成 - 5 分钟快速安装指南

## ⏱️ 总耗时：5 分钟

### 第 1 步：获取 API 密钥（2 分钟）

1. 打开 https://dashscope.aliyun.com/
2. 使用手机号或微信登录/注册
3. 左侧菜单 → "API Key 管理"
4. 点击 "新增 API Key"
5. 复制生成的密钥（以 `sk-` 开头）

> **💡 提示**：新用户会获得 100 万 tokens 的免费额度，足够 500 次完整测试

### 第 2 步：配置密钥（2 分钟）

1. **打开扩展程序**

   - 在浏览器右上角找到"Web 功能自动化测试"图标

2. **点击"⚙️ Qwen 设置"按钮**

   - 位置：popup 页面右上角

3. **输入 API 密钥**

   - 在文本框中粘贴刚刚复制的密钥
   - 格式：`sk-xxxxxxx...`

4. **测试连接**

   - 点击 "🧪 测试连接" 按钮
   - 等待 2-3 秒
   - 看到绿色 ✓ 提示表示成功

5. **保存设置**
   - 点击 "💾 保存设置" 按钮
   - 页面会自动关闭

### 第 3 步：开始使用（1 分钟）

1. **回到 popup 界面**

   - 输入要测试的网址
   - 点击 "▶ 开始测试" 按钮

2. **AI 功能会自动启用**
   - ✅ 智能分析页面
   - ✅ 生成真实测试数据
   - ✅ 自动处理错误
   - ✅ 生成智能报告

## ❓ 常见问题

### Q: 密钥格式不对？

**A**: 确保复制的密钥以 `sk-` 开头，不要包含空格

### Q: 测试连接失败？

**A**:

1. 检查网络连接
2. 确认密钥正确
3. 确认密钥未过期
4. 检查 API 额度是否用完

### Q: 看不到"⚙️ Qwen 设置"按钮？

**A**: 刷新扩展程序，或清除浏览器缓存

### Q: 费用会很高吗？

**A**:

- 新用户有 100 万 tokens 免费额度
- 足够进行 ~500 次完整测试
- 实际成本：约 ¥1.5 元/100 次测试

### Q: 数据安全吗？

**A**:

- ✅ API 密钥仅保存在浏览器本地
- ✅ 不会上传到任何第三方服务
- ✅ 可随时删除或更新
- ✅ 支持禁用 Qwen 功能
