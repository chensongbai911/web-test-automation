### 快速参考

**修复前错误**：

```
SyntaxError: '[aria-*]' is not a valid selector
```

**修复后代码**：

```javascript
Array.from(document.querySelectorAll("*")).filter((el) => {
  return Array.from(el.attributes).some((attr) =>
    attr.name.startsWith("aria-")
  );
}).length;
```
